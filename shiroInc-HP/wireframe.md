
**5問の内容（参考）**

```
Q1. 誰に届けたいですか？
    [ 新規顧客を獲得したい ]
    [ 既存顧客との関係を深めたい ]
    [ 採用・ブランディングしたい ]

Q2. 今の一番の課題は？
    [ そもそも認知されていない ]
    [ 来るけど問い合わせが来ない ]
    [ ブランドが古くなってきた ]

Q3. 商材・サービスの性質は？
    [ 説明が必要な複雑なもの ]
    [ 一目でわかるもの ]
    [ 体験してもらわないと伝わらない ]

Q4. 理想の印象は？
    [ 洗練・上品 ]
    [ 親しみやすい・温かい ]
    [ 革新的・先進的 ]
    [ 信頼・安心・誠実 ]

Q5. 今のサイトの状態は？
    [ ほぼゼロから作りたい ]
    [ リニューアルしたい ]
    [ 部分的に改修したい ]
```
